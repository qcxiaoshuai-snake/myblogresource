### 3. 功能性链接 

点击后，触发某个功能

- 点击后触发执行JS代码

```html
    <a href="javascript: alert('你好')">弹出，你好</a>
```
- 发送邮件 mailTo

```html
    要求用户计算机上安装有邮件发功软件

    <a href="mailto:1669932644@qq.com">发邮件</a>
```

- 拨号，拨打电话 tel

```html
    要求用户计算机安装有拨号软件

    手机端不需要拨号软件，直接放置如下代码

    <a href="tel:18660293796">打电话</a>
```

## target 属性

表示跳转窗口位置

- target=“_self”  当前窗口打开
- target="_blank" 在空白窗口打开
